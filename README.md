
## CARA INSTALL DI TERMUX
```bash
> pkg install nodejs && pkg install git
> git clone https://github.com/YuzzuKamiyaka/bitch-bot
> cd bitch-bot
> bash install.sh
> npm start/node index.js
```
## CARA INSTALL DI LAPTOP
```bash
> git clone https://github.com/YuzzuKamiyaka/bitch-bot
> cd bitch-bot
> npm i
> npm start/node index.js
```
